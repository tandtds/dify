## Type

## Author

## Installation

### From Dify Marketplace
1. Open Dify
2. Navigate to Plugins
3. Search for "archon_agents"
4. Click Install

### From Local File
1. Download the `.difypkg` file
2. Open Dify → Plugins → Upload from local
3. Select the file and install

### From GitHub (if published)
1. Open Dify → Plugins → Install from GitHub
2. Enter repository URL
3. Install
